Contributing to netCDF-Java/CDM and TDS projects

Ways to Contribute

There are many ways to contribute to the netCDF-Java/CDM and TDS projects. Here are a few:

  • Submit bug reports. (Reference: Unidata Support process; some "How to submit bugs" docs)
  • Submit feature requests (Reference: Uni Support process; email lists)
  • Submit patches to the source code
  • Contribute plug-in software
  • Contribute content for the documentation.
